# Minha Vida — Site Pessoal

Projeto criado com **React + Vite** e **CSS puro**,
no estilo de uma landing page, mas com informações sobre a minha vida.

## Como rodar o projeto

1. Instale as dependências (precisa ter o [Node.js](https://nodejs.org) instalado):

   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```

3. Abra o endereço que aparecer no terminal (algo como `http://localhost:5173`).


## Gerar versão final (build)

Quando quiser gerar os arquivos prontos para publicar:

```bash
npm run build
```

Os arquivos finais aparecerão na pasta `dist/`.
